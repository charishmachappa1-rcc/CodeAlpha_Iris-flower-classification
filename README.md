# CodeAlpha_Iris-flower-classification
🌸 Iris Flower Classification using Machine Learning
📌 Overview
This project builds a machine learning model to classify iris flowers into:
Setosa
Versicolor
Virginica
based on flower measurements like sepal and petal dimensions.
🚀 Tech Stack
Python
Scikit-learn
Matplotlib
NumPy
📊 Dataset
Total Samples: 150
Features: 4
Classes: 3
⚙️ Model
K-Nearest Neighbors (KNN)
Simple and effective classification algorithm
📈 Results
✅ Accuracy: 100%
📊 High precision for all classes
📉 Perfect classification (no errors)
📊 Visualizations
Accuracy Graph
Precision Graph
Dataset Distribution (Pie Chart)
Confusion Matrix
Scatter Plot (Petal Length vs Width)
